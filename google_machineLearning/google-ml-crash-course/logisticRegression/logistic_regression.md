### 시그모이드 함수

표준 로지스틱 함수(시그모이드 함수):  $f(x) = \frac{1}{1 + e^{-x}}$ 

![image.png](../../images/sigmoid.png)

입력값 `x`가 증가할수록 시그모이드 함수의 출력값은 `1`에 가까워지지만, 도달하지는 않는다.

로지스틱 회귀 모델의 선형 방정식 : $z = b + w_1x_1 + w_2x_2 + … + w_Nx_N$

- $z$: 로그 오즈, 선형 방정식의 출력
- $b$: 편향
- $w_1, w_2, …w_N$: 가중치
- $x$: feature

시그모이드 함수의 z값으로 전달되어 0~ 1의 확률이 생성된다.

$y' = \frac{1}{1 + e^{-z}}$
